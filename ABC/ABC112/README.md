### A - Programming Education  
[問題ページ](https://atcoder.jp/contests/abc112/tasks/abc112_a)  
- #### 解法  
    nを入力した時点で条件分岐するのが吉  
- #### コード(Python3)  
  
    ```python
    n = int(input())
    if n == 1:
        print('Hello World')
    else:
        print(eval(input()+'+'+input()))
    ```
